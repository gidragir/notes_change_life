# Использование PARA

- **Как работает:** Ты организуешь файлы не по _типу_ (документы, картинки), а по _актуальности_. Закончил проект—перенес папку в Архив. Начал учить Go—создал папку в Ресурсах. Это динамическая файловая система.
- **Результат применения:** "Чистый стол". Ты садишься работать и видишь только то, что нужно сегодня. Нет паралича выбора. Ты знаешь, где лежит конфиг, который нужен для текущего проекта.
