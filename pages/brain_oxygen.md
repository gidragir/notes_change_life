# Кислород для мозга

Наш мозг имеет определенное окно контекста и когда оно уже полностью заполнено, добавление новой информации потребует освобождение этого окна, а значит какая-то определенная или случайная информация будет утерена. Для этого определенные знания лучше обрабатывать и складировать, тем самым у мозга будет больше окна контекста для новый сырых знаний и идей

#### Создаваемая связь

Представим мозг как оперативную память, а заметки это жесткий диск, перенося информацию из оперативной памяти на диск, освобождается место для запуска новых программ
